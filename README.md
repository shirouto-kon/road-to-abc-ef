# Road to ABC

My goals for the AtCoder Beginner Contest (ABC).

## Goals

1. **A, B, C** — solve within 12 minutes.
2. **D, E** — practice them.
3. **F** — solve at least 1 in a real contest within 1 month.

## Implement It Myself

To get past the harder problems, I implement the data structures and
algorithms that show up often — by myself, from scratch.

| Level | Data structure / algorithm to implement |
| ----- | --------------------------------------- |
| D / E | Prefix sum / difference array, two pointers, ~~binary search~~, Union-Find (DSU), BFS / DFS, Dijkstra, basic DP, bitmask DP, modular math & combinations (nCr mod p), coordinate compression |
| F     | Fenwick tree (BIT), segment tree (incl. lazy propagation), advanced DP (digit DP, DP on trees), number theory (sieve, gcd/lcm, fast power), matrix exponentiation, stronger graph algorithms (topological sort, SCC, MST) |
